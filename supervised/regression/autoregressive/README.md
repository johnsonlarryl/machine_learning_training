## About The Project

This project is a practical training tutorial for learning about time series data and autoregressive statistical algorithms.

| Type                | Sub Type       | Algorithm                  |
|---------------------|----------------|----------------------------|

## Usage

In order to view or execute the various notebooks run the following command on any of the sub folders in this directory.

Here is an example to launch the Permutations and Combination Notebook.

```sh
docker compose up -d
```

Open notebooks on your web browser
http://localhost:8888/
